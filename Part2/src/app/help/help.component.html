Help Center

Find answers to common questions and learn how to use the Harvey Norman Inventory System.

📋 Getting Started

  • Use the Manage page to add new products or edit existing ones.
  • The List page shows all inventory items. You can edit or delete each product.
  • Use the Search page to quickly find products by name or description.
  • Your data is stored locally in this demo version. In production, it would connect to a real database.

❓ Frequently Asked Questions

How do I add a product?

Go to the Manage page, fill in the product details (name, price, quantity, etc.), and click "Save".

Can I edit a product after saving?

Yes. On the List page, click the "Edit" button next to the product, update the information, and save.

How do I mark a product as "popular"?

When adding or editing a product, check the "Popular" checkbox. Popular items will be highlighted on the list page.

Is there a search function?

Yes, go to the Search page and type any keyword. The system will filter products that match.

What if I delete a product by mistake?

Deletions are permanent in this demo. In a real system, there would be a confirmation dialog and possibly an undo feature.
